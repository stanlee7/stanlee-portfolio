# SNS 콘텐츠 운영 가이드

목표: **"AI로 실서비스를 만드는 빌더"** 포지셔닝으로 LinkedIn·스레드·X 팔로워를 쌓는다.
원칙: 모든 포스트는 포트폴리오 사이트의 프로젝트 페이지로 링크를 모은다. 링크를 붙이면 OG 카드(스크린샷+제목)가 자동으로 뜬다.

## 기본 포스트 템플릿 (3단 구성)

```
[훅 — 문제 한 줄]
수료증 300장을 손으로 만들던 분을 봤습니다.

[과정 — 어떻게 만들었나, AI 도구 언급]
한/글 COM 자동화 + Claude Code로 일주일 만에
엑셀 명단 → HWP/PDF 일괄 생성 도구를 만들었습니다.
막힌 곳: 백신 오탐, 한/글 버전 분기. 이렇게 풀었습니다: …

[결과 + 링크]
지금 무료로 배포 중입니다.
→ https://stanlee-portfolio.vercel.app/projects/hwp-batch
```

- **LinkedIn**: 과정·교훈 중심으로 길게. "비개발자/현업 분들도 이렇게 할 수 있습니다" 각도가 잘 통함.
- **스레드/X**: 훅 1줄 + 스크린샷 + 링크로 짧게. 숫자(300장, 4시간→15분, 일주일)를 훅에 넣을 것.

## 훅 문장 예시

- "수료증 300장, 클릭 한 번에 만들기까지 일주일 걸렸습니다." (hwp-batch)
- "소방 점검 보고서 작성 4시간을 15분으로 줄였습니다. 코드 한 줄도 외주 없이." (fireguard)
- "혼자서 Play스토어에 앱을 올리면 배우는 것들." (ythink)
- "포토샵 없이 인물 뒤에 텍스트 넣기 — AI 누끼로 1초." (text-behind-image)
- "서버 없이 브라우저에서 LLM을 돌려봤습니다. 결론:" (designer-assistant)
